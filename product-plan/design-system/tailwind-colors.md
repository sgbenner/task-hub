# Tailwind Color Configuration

## Color Choices

- **Primary:** `indigo` — Used for buttons, links, active tab indicators, checkboxes, key accents
- **Secondary:** `amber` — Used for completed count badges, warm highlights
- **Neutral:** `stone` — Used for backgrounds, text, borders (warm gray)

## Usage Examples

**Primary button:**
```
bg-indigo-600 hover:bg-indigo-700 text-white
```

**Primary text link:**
```
text-indigo-500 hover:text-indigo-700 dark:hover:text-indigo-300
```

**Active tab indicator:**
```
border-indigo-500 text-indigo-600 dark:text-indigo-400
```

**Active badge (e.g., task count):**
```
bg-indigo-100 text-indigo-600 dark:bg-indigo-950 dark:text-indigo-400
```

**Secondary badge (e.g., completed count):**
```
bg-amber-100 text-amber-600 dark:bg-amber-950/60 dark:text-amber-500
```

**Neutral text:**
```
text-stone-700 dark:text-stone-300
```

**Neutral muted text:**
```
text-stone-500 dark:text-stone-400
```

**Neutral border:**
```
border-stone-200 dark:border-stone-800
```

**Neutral background:**
```
bg-stone-50 dark:bg-stone-950
```

**Card/surface:**
```
bg-white dark:bg-stone-900 border border-stone-200 dark:border-stone-800
```
